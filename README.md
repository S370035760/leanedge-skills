## 使用效果展示

### 场景1：新建水果包装车间

**你只需说：**
> "我要建一个300㎡的水果包装车间，目标产量10000份/天，请帮我设计布局。"

**AI输出：**
```
✅ 2D平面布局图（标注设备编号、物流动线、尺寸）
✅ 3D效果图（掀顶剖切视图，展示内部布局）
✅ 设备清单（清洗机、切割机、包装机等）
✅ 功能分区（原料区、清洗区、切割区、包装区、冷藏区）
✅ 产能核算（理论产能、实际产能、利用率）
✅ 瓶颈分析（清洗工位瓶颈、改善建议）
✅ 风险识别（产能不足、冷链断链、交叉污染）
✅ 改善建议（1周/1月/3月行动计划）
```

**你只需说：**

```
✅ 风险识别：物流交叉、食品安全风险
✅ 改善建议：增加清洗槽、优化物流动线
```

### 场景3：订单排产

**你只需说：**
> "今天有3个订单，请帮我排产。"

**AI输出：**
```
✅ 订单释放判断：ORD001可排、ORD002可预排、ORD003可排
✅ 资源匹配：6条产线、30人、产能1200份/小时
✅ 排产计划：时间段×产线任务表
✅ 风险识别：ORD002交期紧张、人力不足
```

A：可以。AI会按标准厂房假设设计，但建议补充图纸后重新设计。

A：方案包含老板汇报版摘要，用非专业语言概括核心结论。

**Q：方案不准怎么办？**
